# Mathematical-Modeling
## 数学建模代码库
本仓库对自己参加过的数学建模比赛代码做一个记录。  
### 内容
- pandas、numpy、matplotlib、seaborn等数据分析及可视化库的应用。
- 一些LSTM等简单的模型代码。  
- 文件夹中包括图片和部分数据。
